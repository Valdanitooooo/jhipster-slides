JHipster 是什么?

Java Hipster

JHipster是一个免费的开源应用程序生成器，用于使用Angular或React（JavaScript库）和Spring Framework快速开发现代Web应用程序和微服务。

JHipster吸引我们的地方

  • 开箱即用
  • 集成了最新的前后端框架
  • 方便持续集成
  • 社区活跃

前端技术栈

Yeoman, Webpack, Angular, React, Bootstrap

NPM

  • 快速可靠的依赖管理
  • 用于安装和运行所有客户端工具
							npm install -g generator-jhipster

Yeoman

提供应用程序生成器

Yeoman工作流包括三种类型的工具
  • 脚手架工具（yo）
  • 构建工具（Gulp，Grunt等）
  • 包管理器（如npm和Bower）

Webpack

模块打包机

  • 编译、缩小、优化前端
  • 编译和热重新中加载angular/react+typescript的基本工具
  • 允许在客户端进行高效的生产构建

BrowserSync

应用程序更改的热重加载

  • 修改代码时自动刷新浏览器
  • 实时同步多个浏览器，因此可以立即检查更改
							npm start

Jest

对javascript代码运行单元测试

  • 与jsdom（一种虚拟dom技术）一起工作
  • 非常快，可以在后台连续运行v
							npm test

Angular and React

最流行的javascript框架

  • 强大且易于学习
  • 数据绑定，表单验证，i18n…开箱即用

Bootstrap

优秀的css/javascript框架，用于响应迅速的web应用程序

  • 所有常用的css和html元素
  • 大量的javascript组件：菜单、导航栏、下拉框、警报……
							
							

								Hello, world!

服务器端技术栈

Maven, Spring, SpringBoot, SpringCloud

Maven | Gradle

最流行的java构建工具

  • 与主要IDE集成
  • 丰富的插件生态系统，包括Spring Boot和Yeoman
				                            
				./mvnw spring-boot:run
				                            
				                            
				./gradlew bootRun

Spring Boot

Java应用程序的标准

  • 简化应用程序编码的ioc、aop和抽象
  • 开箱即用， 快速轻巧，功能强大
  • 热加载应用程序
				                            
				@Service
				@Transactional
				public class UserService {
				
				    @Autowired
				    private UserRepository userRepository;
				
				}

Liquibase

数据库更新变得容易

  • 处理更新数据库架构
  • 很适合spring和jpa
  • 需要对数据库和orm有很好的了解

支持多种认证机制

JWT，Session， OAuth 2.0/OIDC

  • JWT - 无状态，带令牌
  • Session - 有状态的、默认的spring安全机制
  • OAuth 2.0/OIDC - 有状态，与Keycloak和Okta一起使用
  • JHipster User Account and Authentication (UAA)

支持多种数据库

常用关系型和非关系型数据

  • SQL (H2, MySQL, MariaDB, PostgreSQL, Oracle, MSSQL)
  • MongoDB
  • Cassandra
  • Couchbase

支持多种缓存技术

  • ehcache
  • Caffeine
  • Hazelcast
  • Infinispan
  • Memcached

监控

生成生产指标

  • 监控jvm、应用服务器、spring beans、缓存等等！
  • JMX或Graphite报告
							
								@Timed
								public void saveAccount(@RequestBody UserDTO userDTO) {
									userService.updateUserInformation(userDTO);
								}

其他

  • Elasticsearch
  • Spring Websocket
  • OpenAPI-generator

微服务的技术栈

Netflix OSS

Netflix OSS

Netflix OSS 为微服务提供了出色的工具和框架

  • 使用Netflix Zuul或Traefik的 HTTP路由
  • 使用Netflix Eureka或HashiCorp Consul进行服务发现
  • Ribbon - 负载平衡

Istio

  • 流量管理 ：超时，重试，负载均衡
  • 安全性： 最终用户身份验证和授权
  • 可观察性： 跟踪，监控和记录

Demo 演示

  • 构建单体应用程序
  • 构建微服务应用程序
  • 持续集成

构建单体应用程序

安装Yeoman和generator-jhipster

				                            
				npm install -g yo
				npm install -g generator-jhipster

创建应用程序

							
							mkdir jhipster-learning
							cd jhipster-learning/
							jhipster

启动后端服务

							
							./mvnw

热加载启动前端

							
								npm start

生产环境启动方式

							
								./mvnw -Pprod package
								docker-compose -f src/main/docker/mysql.yml up -d
								java -jar target/*.jar

总结

  • jhipster大大缩减了项目初期的时间成本
  • CI/CD友好，开发者可以专注于业务逻辑的实现
  • 多种可靠且优秀组件的结合提高了整个团队的工作效率
